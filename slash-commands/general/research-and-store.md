Research a topic using Perplexity and store findings in the vault.

Workflow:
1. Ask user for research topic and depth (quick/deep)
2. Use appropriate Perplexity MCP tool:
   - Quick research: `mcp__perplexity__search`
   - Complex analysis: `mcp__perplexity__reason`
   - Comprehensive: `mcp__perplexity__deep_research`
3. Create a well-formatted research note in Articles/ with:
   - Summary (3-5 bullet points)
   - Key findings
   - Sources and links
   - Related concepts
   - Tags for discoverability
4. Ask if user wants to:
   - Store key insights in Mem0 for long-term memory
   - Link to existing notes
   - Create follow-up research tasks
5. Use Obsidian semantic search to suggest related existing notes

Store research methodology in Mem0 for future reference.
