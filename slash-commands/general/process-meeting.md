Process the most recent Granola meeting note and create action items.

Steps:
1. Find the most recent meeting note in Networking/ or Granola/ folders
2. Extract key decisions, action items, and follow-ups
3. Identify any people mentioned who should be added to person/* tags
4. Create a summary with:
   - Meeting overview (2-3 sentences)
   - Key decisions made
   - Action items with owners (if mentioned)
   - Follow-up topics for next meeting
5. Offer to create follow-up tasks in the relevant project folder

Use semantic search to find related previous conversations with the same participants.
