Create a new project folder structure with standard organization.

Ask the user:
1. Project name
2. Project type (client work, collaboration, personal, education)
3. Initial notes to include

Then create:
```
[Project-Name]/
  ├── CLAUDE.md (project-specific context)
  ├── README.md (project overview)
  ├── meetings/ (meeting notes)
  ├── daily-progress/ (daily summaries)
  ├── resources/ (supporting materials)
  └── Index.md (main entry point)
```

Initialize CLAUDE.md with project-specific context template.
Add project to vault index.
Offer to create initial meeting note or project plan.
